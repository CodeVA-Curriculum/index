Use simple `print` commands to make sure your Python environment is working correctly and you can successfully run code. Given a file called `hello.py` (or whatever) with the following contents:

```python
print("Hello, world!")
```

Run the file using your IDE or by running the following command from the directory containing your `hello.py` file:

```
python hello.py
```
:::
