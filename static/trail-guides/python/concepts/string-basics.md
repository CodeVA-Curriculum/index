## Formatted String

```python
name = "Jon"
print(f"hello, {name}")
```

## Concatenation

```python
first_name: "Jon"
last_name: "Stapleton"

print(first_name + last_name)
```

```python
number_1 = "5"
number_2 = "3"

print(number_1 + number_2)
```
