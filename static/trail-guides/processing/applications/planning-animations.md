Animations can be complicated because they often include multiple elements changing position, moving, and appearing on the canvas. This can be complicated enough to code in and of itself however it becomes even more difficult when there’s no plan created prior to coding the animation.

Creating a mental image of the animation is a good first step. However that step is often not sufficient to develop an animation, especially an animation with multiple moving parts and elements coming and going at different times.

## Storyboarding

One of the most common approaches to animation planning is **storyboarding**.  This approach uses tiles (or "panes") to show what each part of a scene will include.  Storyboards are often created by hand with artists sketching out each scene on a separate tile using pen and paper, but they can also be created digitally. There are many free digital tools available for this purpose.

Here is a simple storyboard template which you can copy and paste into any word processing software:
