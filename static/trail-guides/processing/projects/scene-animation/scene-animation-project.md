TODO: During the Scene Animation Unit, you used variables and the `draw()` method to manipulate shapes to create animated scenes.  For this project, you will create an animation that could be used as an introduction scene or as a cutscene for a video game.  

You will start by creating a storyboard to plan the design and object movement for the animation.  You will need to identify the different components needed for the scene and the properties of those components that will need to change.  

You will then need to create those components with Java Processing and use variables to control the properties of the components so they are able to change when used with the `draw()` method.
